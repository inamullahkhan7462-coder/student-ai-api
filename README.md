# 🎓 Student AI Management System - Backend

A high-performance FastAPI & PostgreSQL backend with integrated Google Gemini AI for automated academic counseling.

## 🚀 Live Links
- **Interactive API Documentation:** [Swagger UI](https://inam-student-api.onrender.com/docs)
- **Live Frontend Hub:** [Student Hub App](https://id-preview--3d17dcfb-58d1-433a-96df-400ed8ee01e0.lovable.app/)

## 🛠️ Tech Stack
- **Framework:** FastAPI
- **Database:** PostgreSQL (with SQLAlchemy ORM)
- **AI Engine:** Google Gemini (2.0/2.5 Flash)
- **Deployment:** Render (Cloud Service)

## 🔑 Key Features
- **CRUD Operations:** Full management of student records.
- **AI Advice Engine:** Generates personalized GPA-based counseling using LLMs.
- **Real-time Analytics:** Optimized database queries for GPA statistics.
- **CORS Enabled:** Securely configured for frontend cross-origin requests.
